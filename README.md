# Dreamscape
Dreamscape is a second-year end-of-the-year project. Done in C++ with SFML

"Help Canni fight his nightmares in three different gameplays. Combine fast-paced 2D platformer, action-rpg and shoot'em up mechanics to defeat the boss ! 

Here is the link to the trailer :  https://www.youtube.com/watch?v=p5G18yWtx4Y

Controls : 

Use ZQSD/WASD bindings. Better played with a xbox controller.
E to interact. 

You can rebind them in the settings.
